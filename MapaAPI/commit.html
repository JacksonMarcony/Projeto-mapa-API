Comentários sobre o projeto voltar ao Jack's World Place Searcher
  • sequência lógica das funções utilizadas no projeto:
    utilizadas 8 functions e 3 arrow function, são elas em sequência:
    Primeiramente foi usado uma função,fCidade(), para pegar o valor digitado pelo usuário no input, e usei um return "cidade" para poder usar o valor dessa função depois...
    Depois foi utilizado uma outra função, dados(), onde é realizada a requisição à API do mapbox.com, então eu pego a função fCidade() para concatenar na url que é usada como "endereço" dos dados a serem requisitados, utilizo essa linha: *requisicao.open('GET', url, true);*, para abrir a conexão e o "true" para determinar que será assíncrona... resaltando que dentro dessa function existe outra function, porém essa é uma void, function(e)...
    A terceira function, valor(), serve para percorrer a resposta da requisição, para isso é feito o uso de um for, que percorre features atrás das coordenadas e o nome dos locais achados... E quando ele acha as coordenadas e os nomes de cada um dos 5 primeiros resultados encontrados, esses valores são passados diretamente como parâmetro para a function inserirItem().
    A quarta function, create(), está apenas criando uma div e um ol, com createElement, essa div e ol são usadas para se jogar os nomes dos locais achados da requisição. Assim eu torno a "var info" filha da div "principal" que já está fixada no documento HTML(index.html), e torno a "var ol" filha da div "info".
    A quinta function, inserirItem(nome_do_item, id, lt,lg), está recebendo os valores passados por parâmetro da function valor(), além disso nela são cridas e geradas as li que estão com o conteúdo dos nomes dos lugares. nessa function, eu utilizei de uma função, String(), para transformar os valores das coordenadas em string, e então uso uma variável "map" para guardar a função concatenada em forma de string com as coordenadas(lt e lg). após esse processo basicamente eu uso um onclick atribuido a 'nova_link', chamando map, que armazena a funtion 'gerarMapa(coord)'.
    A próxima function, Mapa(), tem a função de usando createElement, criar uma div com o id "mapid", e a torna filha da div "MapaCentral" que já está criada no html. Também é agregado um style.width e height para dar tamanho ao mapa quando for carregado na tela.
    A sétima function é uma arrow function, const clear = () => {}, ela tem como única função limpar a div 'principal', onde fica os nomes dos locais, isso para que possa pesquisar locais várias vezes sem ser criadas um bucado de div, li, etc.
    A oitava function, também é uma arrow function, const limparMapa = () => {}, ela tem um objetivo parecido com a anterior, a diferença é que ela limpa a div "MapaCentral", para que se possam ser gerado mapas de varios locais sem ficar dando erro da API "container já foi inializado".
    A nona function é a ultima arrow function, const gerarMapa = (lat, long) => {}, quando ela é chamada outras duas functions também são chamadas (limparMapa() e Mapa()), após esse processo é gerado um mapa ,referente ao valor dos parâmetros lat e long, na div 'mapid'.
    A última function, main(), ela tem a função de chamar as outras functions necessarias quando acontecer uma ação de click no button.
